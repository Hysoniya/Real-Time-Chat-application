A real-time chat application built using React, Node.js, Express, and Socket.IO that enables seamless, low-latency communication between users.

---

## 🚀 Getting Started

Follow the steps below to run the project locally.

---

## 📦 Frontend Setup

1. Navigate to the frontend directory:

```bash
cd public
Install dependencies:
npm install
Start the frontend:
npm start

The frontend will run on:
👉 http://localhost:3000

⚙️ Backend Setup
Navigate to the server directory:
cd server
Install dependencies:
npm install
Start the backend server:
nodemon index.js
🛠️ Tech Stack
Frontend: React.js, JavaScript (ES6+), CSS
Backend: Node.js, Express.js
Real-Time Communication: Socket.IO
Database: MongoDB
Authentication: Bcrypt
✨ Features
Real-time messaging with low latency
Secure user authentication
Responsive UI
Scalable backend architecture
📌 Note

Make sure both frontend and backend servers are running simultaneously for proper functionality of the application.
